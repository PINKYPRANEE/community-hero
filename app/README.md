## 📌 Problem Statement
Communities frequently face issues such as potholes, water leakages, damaged streetlights, waste management concerns, and public infrastructure challenges. Reporting these issues is often fragmented, difficult to track, and lacks transparency.

---

## 💡 Solution
**CommunityHero** is an AI-powered civic platform that enables citizens to:
- 📸 **Report** community issues with photos
- 🤖 **AI auto-detects** the issue category, title, description and severity
- 📍 **GPS auto-detects** exact location
- 🗺️ **View all issues** on a real Google Map
- 👍 **Upvote** issues to show community support
- 📊 **Track progress** via live dashboard
- 🏆 **Gamification** via leaderboard

---

## ✨ Features

### 🤖 AI-Powered
- **Image Analysis** — Upload a photo, Gemini AI automatically detects the issue type
- **Auto-fill** — AI writes the title, description and severity automatically
- **Smart Categorization** — Pothole, Water Leak, Streetlight, Waste, Tree Fall, Infrastructure

### 🗺️ Real-time Mapping
- Google Maps integration with live issue pins
- GPS auto-detection for exact location
- Works from any city in the world

### 📊 Community Dashboard
- Real-time statistics
- Issues by category with progress bars
- Resolution rate tracking
- Recent activity feed

### 🏆 Gamification
- Community upvoting system
- Leaderboard for top reporters
- Issue status tracking (Reported → In Progress → Resolved)

### 🔍 Smart Filtering
- Filter by category
- Filter by status
- Real-time search
